# Review SFDC Rules

**Plugin skill:** `oxels-review-sfdc-rules`

Review the current SFDC contract upload rules and, if needed, manage them: add a new rule with clear PASS / FAIL / NOT_APPLICABLE conditions; edit an existing rule (rename, re-scope, rewrite, or change blocking — each edit saves a new version you then promote live); turn rules on or off; roll back to an earlier version; and review the change history / audit log — what changed, when, and when a rule was activated or deactivated.
